## Setup
**Before running python server:**

Install dependencies:
```shell
pip3 install -r requirements.txt
pip install psycopg2
```

## Usage
Run the following in console:
```shell
python main.py
```

## How to run on mac (M1)

```
pip install virtualenv
```

```
virtualenv flask
```

```
cd flask
```

```
source bin/activate
```

```
pip install flask
```

## Run
```
python main.py
```
