With keyboard buliding there are really only six major component that arenecessary for a standard build. These parts are as follows; A PCB, A Case, A Plate, a setof Stabilizers, a set of switches, and a set keycaps. There are of course other components. But those are mainly for aesthetics or for the board to make a better sound. These would be items like pcb-plate foam, pcb-case foam, LED's, LED diffusers, aviator USB cables, gasket mount band, risers etc. We hope that the following part guide will help you on your quest and Happy Building!
